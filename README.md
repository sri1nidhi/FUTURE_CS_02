# Phishing Email Detection & Awareness System

## Objective
Analyze phishing emails and create awareness.

## Tools Used
- Google Header Analyzer
- MXToolbox
- Browser tools

## Approach
1. Collected phishing samples
2. Analyzed headers and links
3. Identified indicators
4. Classified risk
5. Created awareness guidelines

## Outcome
Improved understanding of phishing attacks and prevention techniques.
